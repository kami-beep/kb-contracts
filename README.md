## 📖 What is kb-contracts?

Kb-contracts helps organize and manage agreements within a data and knowledge system. It covers decisions and contracts for things like:

- Events and sessions  
- Summaries and digests  
- Data chunks and manifests  
- Run records and publishing contracts  

It supports systems that use multiple repositories connected through bus-based workflows. This helps teams keep track of how data moves and changes throughout a knowledge pipeline.

Kb-contracts is especially useful if you work with:

- Data contracts and governance  
- Event-driven systems  
- Knowledge management  
- Vector databases and retrieval systems  

You do not need to be a programmer to use it.

Kb-contracts provides an interface to manage knowledge pipeline contracts in an organized way.

### 1. View Your Contracts  
You can see lists of contracts for events, sessions, summaries, and more. Each contract shows basic information like name, date, and related files.

### 2. Add New Contracts  
Create new contracts easily by filling in simple forms. You will enter contract details such as type, participants, and status.

### 3. Edit Existing Contracts  
Update details on contracts when things change. This keeps information accurate and up to date.

### 4. Track Data Flow  
Kb-contracts shows how data chunks and manifests move through your system. This helps ensure data governance and compliance.

### 5. Export or Share Contracts  
Export contracts as files or share them via email. This helps coordinate across teams.
